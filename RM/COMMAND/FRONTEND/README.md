install heroUI-CLI 
`npm install -g heroui-cli`
Setup project
`heroui init`

Change directory
`cd <project_name>`
Install dependency
`npm install`
Fix dependency error
`npm audit fix --force`
Add all heroUI components
`heroui add --all`

start frontend server
`npm run dev`

install Bootstrap icon
`npm install react-bootstrap-icons`

install jose for JWT
`npm install jose`
